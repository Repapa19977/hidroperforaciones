# Components

Componentes reutilizables del CRM.

- Paneles grandes deben vivir aqui si se usan desde una pagina de `app/`.
- Componentes pequenos de metrica, dialogos y tarjetas se comparten desde aqui.
- Evitar duplicar UI dentro de paginas cuando ya existe un componente equivalente.

Las paginas de `app/` deberian coordinar datos y estado de ruta; los bloques
visuales repetibles deben quedar en esta carpeta.
